# FitPath Fonts

All fonts are loaded via `next/font/google` (no local font files).

## DM Sans — Primary / UI font
- CSS variable: `--font-dm-sans`
- Weights: 300, 400, 500, 600, 700 (normal + italic)
- Usage: body text, UI labels, navigation, buttons
- Google Fonts: https://fonts.google.com/specimen/DM+Sans

## Instrument Serif — Heading / Display font
- CSS variable: `--font-instrument-serif`
- Weights: 400 (normal + italic)
- Usage: page headings, hero titles, decorative text (`h1`, `.font-serif`)
- Google Fonts: https://fonts.google.com/specimen/Instrument+Serif

## Geist Mono — Monospace
- CSS variable: `--font-geist-mono`
- Usage: code snippets, `<code>` elements
- Source: bundled with Next.js (`next/font/local` via Vercel)
