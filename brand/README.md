# FitPath Brand Assets

## Folder structure

```
brand/
├── logos/
│   ├── icon.svg      ← App icon (dumbbell on purple #aaa8ff, 32×32)
│   └── favicon.svg   ← Browser favicon (emoji on orange #CF7B4B, 32×32)
└── fonts/
    └── fonts.md      ← Font names, weights, CSS variables, Google Fonts links
```

## Colors (from globals.css design tokens — dark mode)

| Token | Value | Use |
|---|---|---|
| `--fp-accent` | `#aaa8ff` | Primary brand lavender |
| `--fp-accent-2` | `#6b69cc` | Darker lavender |
| `--fp-black` | `#030203` | Background |
| `--fp-surface` | `#111318` | Card surface |
| `--fp-surface-2` | `#2b2e3f` | Elevated surface |
| `--fp-text` | `rgba(255,255,255,0.81)` | Primary text |
| `--fp-text-muted` | `rgba(255,255,255,0.45)` | Muted text |
| `--fp-green` | `#7f9ef8` | Berry blue (progress/stats) |
| `--fp-purple` | `#c1cff8` | Periwinkle |
| `--fp-blush` | `#fad0f3` | Blush pink |
| `--fp-yellow` | `#fffba5` | Soft yellow |
